* ms_rptonation tool:
Files with extension ".crgms" are meant for use with the `ms_protonation` command-line tool.
Make a local copy to your working dir 'some_dir':
    cd some_dir
    cp path_to_clone/MCCE4-Tools/mcce4_tools/tool_param/params.crgms .
    # Edit the file, then run the full command:
    ms_rotonation params.crgms
